#### `Delete` of CRUD 

views.py

```python
def delete(request, pk):
    article = Article.objects.get(pk = pk)
    article.delete() 
    return render(request, 'articles/delete.html', context)
```



#### `Update` of CRUD

views.py

```python
def edit(request, pk):
    article = Article.objects.get(pk=pk)
    context = {
        'article' : article
    }
    return render('articles/update.html', context)

def update(request, pk):
    article = Article.objects.get(pk = pk)
    article.title = request.Get.get('title')
    article.content = request.Get.get('content')
    article.save()
    return redirect(f'/articles/review_detail/{article.pk}/')
```
